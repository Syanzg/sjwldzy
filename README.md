# 基于堆叠LSTM的美股收盘价预测模型
本仓库包含基于LSTM神经网络的美股收盘价预测完整实验代码，覆盖数据预处理、模型构建、超参数调优、性能评估全流程，可直接复现实验结果。

## 一、数据集说明
### 1. 基本信息
- **数据集名称**：Huge Stock Market Dataset（美股价格与成交量数据集）
- **数据源链接**：https://www.kaggle.com/datasets/borismarjanovic/price-volume-data-for-all-us-stocks-etfs
- **时间范围**：2000年至2023年（总计18+年）美股日度交易数据
- **覆盖标的**：8300+ 美股个股及ETF（实验选取苹果公司AAPL）
- **原始特征维度**：日期（Date）、开盘价（Open）、最高价（High）、最低价（Low）、收盘价（Close）、成交量（Volume）
- **实验特征维度**：原始特征 + 衍生特征（MA5、MA10、Price_Range、Volume_Norm），共9维特征

### 2. 预处理方法
1. 缺失值处理：前向填充（ffill）+ 删除剩余缺失值，无缺失值残留；
2. 重复值处理：基于日期去重，清理重复数据12条；
3. 特征工程：构建5日/10日移动平均线、价格波动范围、归一化成交量；
4. 归一化：MinMaxScaler缩放到[0,1]区间；
5. 时序构建：60天时间步长，8:2划分训练集（12876条）/测试集（3219条）。
